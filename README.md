## 📝 Descripción del Proyecto
Este repositorio contiene la presentación y el archivo en LaTeX con la resolución de la tarea  de **5 ejercicios de anualidades** para el curso de Estadística Actuarial 2026-I

El informe incluye el planteamiento, las fórmulas aplicadas y el desarrollo paso a paso de los siguientes problemas:
* **2 Ejercicios de Anualidades Anticipadas** 
* **3 Ejercicios de Anualidades Vencidas** 

---

## 📂 Estructura del Repositorio

* `Ejercicios_Anualidades.tex`: Archivo fuente principal en LaTeX que contiene la carátula, el desarrollo de los problemas y las fórmulas.
* `document.tex`: Archivos de configuración o módulos secundarios empleados en el documento.
* `UNSAAC.png`: Logo institucional utilizado en la carátula del trabajo.
* `.gitignore`: Archivo de configuración de Git para evitar la subida de archivos temporales e innecesarios de la compilación (`.aux`, `.log`, `.synctex.gz`).

## 🛠️ Herramientas Utilizadas
* **LaTeX**: Para la tipografía matemática y estructuración del informe.
* **Git / GitHub**: Para el control de versiones, orden del proyecto y almacenamiento del trabajo.
